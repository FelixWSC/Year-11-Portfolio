Low Poly Character

Overview of the project

For this project I will be creating a low-polygon character for a video game. I'll need to design and create a unique character on paper then model it in blender. I'll then need to create animations and rigging of the character so that I can make it do different movements and animations.


My character is a very short, stubby knight. For this project I really wanted to make a very cartoonish knight character and gained my insparation from random tv shows and medival type video games.

End users

The main people that will use my characters are probably gamers playing a game where the developer included my character. They probably are a younger demographic either a adolescent or teenage audience playing some unique indie medival fighter or exploring title that has a bit of a more unique cartoonish style. They'll probably want to play a unique looking character with advanced, fleshed out movements and animations so that character looks good in motion, action, and fighting scenes.

Requirements

My low-poly character is required to have and need many things. First off all the characters spot such as the torso, upper arms, lower arms, and lower and upper legs sections. It will also need a neck to connect the head to the mainbody along with feet for the character and maybe if I have an excess of time a plume/feather for the top of the knights helmet. The character also needs rigging so that it can carry out movements and animations. Its rigging will be done by rigging it so that its rigged with a single spine bone and other bones can be connected to it.

Specifications

  • Properly rigged with a spine
